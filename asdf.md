### asdf

```bash
asdf help

asdf plugin-add elixir
# add local

asdf list-all elixir
# list all available versions

asdf install elixir 1.10.3
# install this version (also can use latest/master)

asdf local elixir 1.10.3
# use this version 

asdf list elixir
# list all installed versions of elixir

asdf global erlang 21.0.5
# set as global version
```
